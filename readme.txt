This is a plugin to integrate your woocommerce website with refersell. This will help boost your sales.

== Description ==

Boost the sales of your woocommerce website with this plugin. You need to get registered and verified at refersell to use this plugin.
This plugin is presently applicable only for woocommerce websites based in India. We will soon expand to all countries.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Integrate refersell screen to configure the plugin
4. Login to your refersell account by providing your credentials.
5. Click on Create 'Web Services' Page
6. Refresh the page a few times.
7. Select the categories which you want sync with ReferSell Network.
